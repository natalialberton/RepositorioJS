Estrutura de Repetição PARA

A estrutura PARA em JavaScript é uma das principais maneiras de criar loops (ou laços de repetição) que permitem executar um bloco de código várias vezes. Dentro da condição do "for", informamos o valor da variável, a condição para a execução do código e "variável++", para que, toda vez que o código for executado, seja adicionado 1 ao valor atual da variável contadora.
