Estrutura Condicional SE

A estrutura condicional SE no JavaScript permite executar blocos de código com base em condições específicas. De maneira simples, se (if) uma condição é verdadeira, o bloco de código dentro dela será executado. Senão (else), as ações dentro de "if" são puladas e o que está dentro de "else" será executado. É também possível utilizar estruturas "else if" para caso de múltiplas condições diferentes.
