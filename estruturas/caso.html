Estrutura Condicional CASO

A estrutura CASO no JavaScript é usada dentro de uma declaração "switch" para permitir a execução condicional de código com base no valor de uma expressão. O "switch" é uma alternativa ao uso de múltiplas instruções "if" e "else if" e pode tornar o código mais legível quando há muitas condições a serem verificadas.
