Propósito da Atividade

A ideia deste projeto é criar um repositório onde se possa não apenas guardar, como também executar códigos em HTML no próprio site em que eles se encontram. O objetivo principal seria a praticidade para rever os conteúdos, de forma que todos os conceitos básicos de Lógica de Programação fiquem armazenados em um único local, sem que o usuário tenha que abrir pasta por pasta em uma área de trabalho para achar o exemplo de que precisa. Tendo isso em mente, os conteúdos aqui reunidos dizem respeito ao conhecimento que obtive nos primeiros meses em contato com as linguagens utilizadas, e visam oferecer uma pequena base para futuras consultas, se assim forem necessárias.
